# Semperland Persona (libraries project)

This project contains several utilities (libraries) related to the Semperland ecosystem. While this project contains
demo code and implementation, it actually defines useful libraries under addons/, all of them having tha same prefix:
`Semperland.*/`. The idea is for them to copy them and use them in and Semperland project (mainline or third party).

Launching this project requires reading and following the [instructions on dependencies](addons/README.md).
