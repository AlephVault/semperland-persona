# Semperland Persona (libraries project)

This project contains several utilities (libraries) related to the Semperland ecosystem. In particular, managing the
Personas (blockchain identities and the possibility to extend the available citizen assets). While this project contains
demo code and implementation, it actually defines useful libraries under addons/, all of them having tha same prefix:
`Semperland.Persona/` and `Semperland.Persona.*/`. The idea is for them to copy them and use them in and Semperland
project (mainline or third party).

Launching this project requires reading and following the [instructions on dependencies](addons/README.md).
