# Redot / Godot template

This template stands for a Redot game serving as backend and / or front-end of a game.

For this to work, developers typically either:

- Use this directory to hold both client and server code.
- Copy this directory once and then rename this directory as well to have separate applications for client and server.

Either case involves also reading and following the [instructions on dependencies](addons/README.md).
